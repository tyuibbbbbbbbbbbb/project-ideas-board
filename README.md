# לוח הרעיונות - Project Ideas Board

אתר שבו כל אחד יכול לפרסם רעיון לפרויקט שהוא לא יכול/רוצה לפתח בעצמו,
ומפתחים אחרים יכולים לגלוש, להצביע (לייק/דיסלייק) ולאמץ ("יאללה עלי!") רעיונות.

- רעיונות ממוינים לפי ניקוד (לייקים פחות דיסלייקים) - הכי מוצבע למעלה.
- רעיון עם יותר מ-10 דיסלייקים עובר אוטומטית לסטטוס "בבדיקה".
- כשמישהו לוקח רעיון, הכרטיס משנה צבע (ירוק) ומוצג "נלקח ע"י ...".
- מי שאימץ רעיון יכול לסמן אותו כ"בוצע" כשהוא מסיים.
- הרשמה עם שם מלא, שם משתמש בפורום, אימייל וסיסמה.
- אפשר לשמור רעיונות "לעצמי" (רשימה פרטית, לא נראית לאף אחד אחר).

## איך זה בנוי - הכל דרך GitHub, בלי שירותי צד שלישי

| חלק | איפה | תיאור |
|---|---|---|
| האתר (frontend) | **GitHub Pages** (ריפו זה) | קבצי HTML/CSS/JS סטטיים |
| הנתונים (משתמשים + רעיונות) | ריפו **פרטי** נפרד `project-ideas-data` | נשמר כקבצי JSON, עם היסטוריית commits מלאה |
| ה"מתווך" המאובטח | **Cloudflare Worker** (חינמי) | היחיד שמחזיק את מפתח הכתיבה לריפו הפרטי; מטפל בהרשמה/התחברות/כתיבה |

**למה צריך "מתווך"?** אתר סטטי (כמו GitHub Pages) לא יכול להחזיק סוד (כמו מפתח כתיבה
לריפו) בלי שכל מבקר באתר יוכל לגנוב אותו מקוד ה-JS. ה-Worker פותר את זה: הוא מריץ קוד
בצד השרת (חינם, בלי לתחזק שרת בעצמכם) ורק הוא יודע את הסוד.

## הפעלה

1. עקבו אחרי ההוראות ב-[`worker/README.md`](worker/README.md) כדי לפרוס את ה-Worker
   (דורש חשבון Cloudflare חינמי + Personal Access Token מוגבל לגיטהאב).
2. עדכנו את `config.js` עם כתובת ה-Worker שקיבלתם.
3. `git push` - האתר יתעדכן אוטומטית ב-GitHub Pages תוך דקה-שתיים.

## הרצה מקומית לבדיקה

```
npm run dev
```

ואז פתחו: <http://localhost:5500> (ה-Worker חייב להיות פרוס כבר כדי שהתחברות/הרשמה יעבדו).

## מבנה הפרויקט

| קובץ | תיאור |
|---|---|
| `index.html` | מבנה הדף (כותרת, טפסים, כרטיסי רעיונות) |
| `style.css` | עיצוב (ערכת נושא כהה עם זהב) |
| `config.js` | כתובת ה-Worker |
| `app.js` | כל הלוגיקה - הרשמה/התחברות, פרסום רעיון, הצבעה, אימוץ, שמירה |
| `worker/` | קוד ה-Cloudflare Worker (ה"שרת" הקטן שמדבר עם ריפו הנתונים) |

## רעיונות להרחבה עתידית

- מסך ניהול (Admin) לצפייה ברעיונות במצב "בבדיקה" והחלטה אם להחזיר/למחוק.
- תגובות על רעיונות.
- התראות למפרסם כשמישהו מאמץ את הרעיון שלו.
